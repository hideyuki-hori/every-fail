Cloudflare Workersで実現するエッジコンピューティング

2025-06-15
rust wasm

Cloudflare Workersはエッジで動作するサーバーレスプラットフォームです。世界中のデータセンターでコードが実行されるため、ユーザーに最も近い場所から高速にレスポンスを返せます。

workers-rsの導入

workers-rs クレートを使うと、RustでCloudflare Workersのハンドラーを書けます。WASMにコンパイルされてV8のランタイム上で実行されます。

Static Assetsとの組み合わせ

Workers + Static Assetsを使うことで、静的ファイルの配信とWorkerの動的処理を一つのプロジェクトにまとめられます。ルーティングも柔軟に設定できます。

デプロイ

Cloudflare APIを直接叩くことで、CI/CDパイプラインなしでもコマンド一発でデプロイできます。WASMバイナリとstatic assetsを一括でアップロードします。
