Rustのエラーハンドリングパターン集

2025-07-27
rust

Rustでは例外ではなく Result<T, E> 型を使ってエラーを処理します。この仕組みにより、エラーの発生箇所と処理箇所が型レベルで明確になります。

thiserror vs anyhow

ライブラリではthiserrorで構造化されたエラー型を定義し、アプリケーションではanyhowで簡潔にエラーを伝播させるのが一般的なパターンです。

?演算子の活用

? 演算子はResult値をアンラップし、エラーの場合は早期リターンします。From トレイトと組み合わせることで、異なるエラー型間の自動変換も実現できます。

カスタムエラー型の設計

エラーに文脈情報を持たせたい場合は、enumで各バリアントを定義し、Display と Error トレイトを実装します。thiserrorの #[error(...)] マクロで簡潔に書けます。
