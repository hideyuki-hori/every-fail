Rustで始めるWebAssembly入門

2025-06-01
rust wasm

WebAssemblyはブラウザ上でネイティブに近いパフォーマンスを実現する技術です。Rustはそのコンパイルターゲットとして最も人気のある言語の一つです。

環境構築

まず wasm-pack をインストールします。これはRustのコードをWASMにコンパイルし、npmパッケージとして配布できる形にしてくれるツールです。

cargo install wasm-pack

プロジェクト作成

cargo new --lib でライブラリプロジェクトを作成し、Cargo.toml に crate-type = ["cdylib"] を追加します。

JavaScriptとの連携

wasm-bindgen を使うことで、RustとJavaScript間でシームレスにデータをやり取りできます。関数のエクスポートだけでなく、DOM操作やWeb APIの呼び出しも可能です。
