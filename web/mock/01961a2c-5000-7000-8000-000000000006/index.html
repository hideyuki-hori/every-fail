CSSカスタムプロパティでデザイントークンを管理する

2025-07-06
design

デザイントークンはデザインシステムの基盤となる値の集合です。色、スペーシング、タイポグラフィなどの値を一元管理することで、デザインの一貫性を保てます。

CSS Custom Propertiesの利点

CSSカスタムプロパティはネイティブにブラウザでサポートされており、JavaScriptからも参照・変更が可能です。SassやLess変数と異なり、ランタイムで値を変更できます。

テーマの切り替え

:root や data-theme 属性を使って、テーマごとにカスタムプロパティの値を切り替えられます。ダークモードの実装もシンプルになります。

ビルドパイプラインとの統合

デザインツールからエクスポートしたトークンをCSSカスタムプロパティに変換するビルドステップを組み込むことで、デザインとコードの同期が自動化できます。
